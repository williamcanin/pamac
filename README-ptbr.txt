Pamac Installer V1.0

[SOBRE]

Este é um script simples para facilitar a instalação da aplicação 
Pamac (https://aur.archlinux.org/packages/pamac-aur) no Archlinux.

[USANDO]

I) Tenha certeza que você tem uma conexão com a internet razoável.

II) Execute o comando abaixo para descarregar o instalador do Pamac:
   
   $ curl -sL https://git.io/pamac | bash

III) Após o comando ser concluído e o instalador ser descarregado, execute 
o comando abaixo para instalar o Pamac:

   $ bash $HOME/.pamac/install

   Você estará no instalador do Pamac, ele é bem intuitivo e te guiará até 
   o fim da instalação.
